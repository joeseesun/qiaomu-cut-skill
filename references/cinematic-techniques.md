# Cinematic Techniques

这份文件是 qiaomu-cut 的“剪辑语法库”。根据工作流和素材条件选择，不要堆特效。

## 剪辑结构

| 技法 | 用途 | 注意 |
|---|---|---|
| cold open | 先上最强画面/台词 | 适合短视频和英语混剪 |
| hook-card | 文字钩子 | 3 秒内说清价值 |
| montage | 快速建立主题 | 需要音乐节奏 |
| setup-payoff | 铺垫后释放 | 适合故事/观点视频 |
| contrast cut | 对比剪辑 | 前后、旧新、错对 |
| callback | 结尾回扣开头 | 增强完整感 |

## 转场

| 转场 | 适合场景 |
|---|---|
| hard cut | 台词、观点、节奏明确 |
| soft cut | 纪录、课程、人物介绍 |
| match cut | 形状/动作/语义匹配 |
| whip pan | 快速场景切换 |
| luma fade | 电影感、梦境、回忆 |
| mask reveal | 产品、网页、信息图 |
| push/slide | PPT/卡片式演示 |
| glitch | 科技、故障、赛博风，少用 |

## 运镜

| 运镜 | 实现 |
|---|---|
| slow_push_in | 静图或视频裁切缓慢放大 |
| pull_back | 从细节拉到全局 |
| pan_scan | 横向扫过大图 |
| parallax_photo | 前中后景分层位移 |
| handheld_sim | 轻微抖动，不要过量 |
| dolly_zoom_sim | 缩放 + 透视感模拟 |
| map_route | 地图路径动画 |
| screen_zoom | 产品录屏局部放大 |

## 字幕与字体

| 技法 | 用途 |
|---|---|
| bilingual subtitle | 英语学习、访谈、资料片 |
| word highlight | 跟读、重点词 |
| kinetic typography | 情绪、节奏、观点 |
| lower third | 人名、身份、来源 |
| archive label | 年份、地点、出处 |
| safety-area caption | 竖屏平台避免 UI 遮挡 |

字体建议：

- 中文正文：思源黑体 / 苹方 / Noto Sans CJK。
- 中文标题：思源黑体 Heavy / 站酷类标题字，注意授权。
- 英文标题：Inter / SF Pro / IBM Plex Sans。
- 电影感：窄体、衬线、轻字重，避免花字滥用。
- 科普：清晰 sans + 公式字体。

## 音频

- 旁白永远优先清晰。
- 音乐 ducking，不能盖住人声。
- 短视频响度目标可用 -14 LUFS；课程/横屏可用 -16 LUFS。
- 保留 6-12 dB headroom，避免爆音。
- 音效服务于剪辑点，不要每个动作都加。

## 风格克制原则

专业感来自选择，不来自堆叠。一个 scene 最多突出一个主技法：要么转场，要么字幕，要么运镜，要么特效。多个技法叠加时必须有叙事理由。
