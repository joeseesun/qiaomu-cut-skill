# 中文字体主题

按语义选择并检测本机/项目字体；括号内从左到右回退，最后统一到 Noto Sans CJK SC：

- 现代清晰：思源黑体 / Noto Sans CJK SC
- 编辑杂志：思源宋体 / Noto Serif CJK SC
- 温暖人文：霞鹜文楷 / LXGW WenKai
- 科技理性：MiSans / HarmonyOS Sans SC
- 商业醒目：阿里巴巴普惠体 3 / Source Han Sans SC
- 电影标题：优先项目已授权的窄体黑体；无授权则 Source Han Sans SC Heavy
- 轻松口语：PingFang SC / Noto Sans CJK SC
- 复古纸张：项目已授权宋体 / Noto Serif CJK SC

标题与译文不得为了“丰富”随机换字体：一个视频使用一个主题，最多标题/正文两种家族。不得复制或分发系统字体；字体不可用时记录回退结果，缺字即失败。
